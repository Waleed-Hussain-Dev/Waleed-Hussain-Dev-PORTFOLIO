## Overview

This portfolio is a personal website designed to present my work, skills, experience, and contact information in a polished, professional way.

Built with **Next.js 15**, **React 19**, **TypeScript**, and **Tailwind CSS**, the site focuses on performance, accessibility, responsive design, and a refined dark visual style.
